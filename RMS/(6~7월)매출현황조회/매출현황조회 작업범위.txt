작업 범위
1. UI 작업
   1-1) 조회영역 
   1-2) 구글차트플러그인
   1-3) 표데이터 (비영업일 일자, 최대값 최소값, 스크롤)
   1-4) 조회기간 달력 오픈소스 사용 GA와 같도록 라이브러리 재설정
   1-5) 계정리스트 페이지 생성
2. 기능작업
   2-1) 광고유형, 광고매체 + - 처리
   2-2) 소속 (부문, 팀, 담당자) 함수 사용
   2-3) 광고주 필터 생성 (광고주명, 사업자번호, 광고계정)
   2-4) 조회유형 별 설정
   2-5) 달력 전송 값 확인
   2-6) 조회시 파라미터 적용된 쿼리 도출 확인
   2-7) 측정기준별 표데이터 변경 (기간, 부문, 광고주)
   2-8) 측정기준별 표데이터 다운로드 (기간, 부문, 광고주)
   2-9) 측정기준 중 광고주별 Table은 페이징처리
   2-10) 표데이터 숫자 클릭시 계정리스트 팝업실행
   2-11) 계정리스트 다운로드 기능

미작업 범위
1. 데이터 작업
    1-1) 조회필터에 따른 새 쿼리 작성 후 데이터 도출 (조회필터 별, 측정기준 별 쿼리 모두 작성)
    1-2) 도출한 데이터 가공 후 Table 양식에 맞게 화면에 표출 (비교대상 조회 포함)
    1-3) 측정기준별 (기간, 부문, 광고주) 모두 적용 (각각의 쿼리가 필요)
    1-4) 도출한 데이터 가공 후 구글차트에 적용 (json형태로 저장)
    1-5) 계정리스트 데이터 도출쿼리 작업 후 데이터 가공하여 table 화면에 표출
